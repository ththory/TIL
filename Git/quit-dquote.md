## Dquote>에서 빠져나가기

깃 메시지를 작성하다가 엔터를 잘못 입력하여 dquote에 들어갔다.

```shell
git commit -am "A
dquote> message...
```

입력을 취소하기 위해서 `control + G` 를 눌러 dquote에서 빠져나왔다.



+) 추가로 "" 내에 느낌표(!)를 포함한다면 dquote에 빠지는데

이 때는 쌍따옴표("")가 아닌 따옴표('')를 사용하거나 \\!를 사용하면 된다고 한다.
